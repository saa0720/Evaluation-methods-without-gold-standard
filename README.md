This repository provides example R code for statistical methods for evaluating diagnostic tests in the absence of a gold standard, accompanying a systematic review of latent variable approaches in this setting. Six scenarios are covered:

$\bullet$ realdata_1test.R: single-test evaluation

$\bullet$ realdata_2tests_CI.R: two-test model under the conditional independence assumption

$\bullet$ realdata_2tests_fixedeffects.R: two-test model with a fixed-effects structure

$\bullet$ realdata_2tests_randomeffects.R: two-test model with a random-effects structure

$\bullet$ realdata_2tests_2populations.R: two-test model with two populations

$\bullet$ realdata_3tests_CI.R: three-test model under the conditional independence assumption

Code developed in support of: Sun A, Zhou X H. Estimation of diagnostic test accuracy without gold standards. Statistics in Medicine, 2025.
